Comment intégrer TensorFlow

Publié le 10 novembre 2025
Par Admin
Machine LearningTensorFlow
K
Kernel Team Auteur
Il y a 2 jours

Dans cet article on va voir comment intégrer TensorFlow dans un projet web et localement sur votre machine.

Prérequis
  • Python 3.8+
  • Un environnement virtuel (venv)
  • pip à jour
Installation
pip install --upgrade pip python -m venv venv source venv/bin/activate # sous macOS / Linux venv\Scripts\activate # sous Windows pip install tensorflow
Exemple minimal
import tensorflow as tf model = tf.keras.Sequential([ tf.keras.layers.Dense(10, activation='relu', input_shape=(5,)), tf.keras.layers.Dense(1) ]) model.compile(optimizer='adam', loss='mse') print(model.summary())
